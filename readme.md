# 前端网站项目

## 项目概述

这是一个前端网站项目，包含多个网页演示和完整的应用页面。

## 主要内容

### 1. 星辰学院官网 (school/)
完整的学院官方网站页面，包含：
- 顶部导航栏和顶部信息栏
- 轮播横幅区域
- 数据统计展示（建校年数、在校学生、教职员工、科研成果）
- 快捷服务入口（招生信息、教务系统、就业服务等）
- 新闻动态和通知公告
- 学院简介
- 院系展示（计算机科学、商学院、艺术设计等6个院系）
- 校园风采图库
- 响应式页脚设计

### 2. 翻书效果演示 (wab.html)
使用 jQuery 和 turn.js 插件实现的图片翻书效果：
- 展示 photos/ 文件夹中的图片
- 可交互的翻页效果


### 3. 未完成页面 (wab1.html)
简单的 HTML 骨架，待完善

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **jQuery 3.6.0** - JavaScript 库
- **turn.js 4.1.0** - 翻书效果插件
- **Font Awesome 6.4.0** - 图标库
- **Noto Sans SC** - 中文字体

## 使用方法

### 查看学院官网
直接在浏览器中打开 `school/index.html` 文件

### 查看翻书效果
1. 确保 `photos/` 文件夹中有图片文件
2. 在浏览器中打开 `wab.html`
3. 点击页面可进行翻页

## 特性

- 响应式设计，支持桌面端和移动端
- 现代化的 UI 设计
- 流畅的动画效果
- 渐变色彩方案
- 卡片式布局
